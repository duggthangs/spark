Plan Review: Phase 3

CLI Runtime & Integration

Context & Goals

The goal is to transform the project from a "library + playground" into a "runnable application". The IAEE CLI will allow users to run iaee run experience.json, which will start a local server, open the browser, and capture the user's interactive session.

Key Components

  • CLI Entry Point: bun cli/index.ts
  • Local Server: Bun.serve (UI + API)

Objectives

  • Serve static UI assets
  • Expose API for experience JSON
  • Handle user submission
